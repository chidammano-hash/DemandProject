# Interview Call Playbook - Senior Director, Data Engineering (Healthcare/Pharma)

## 1) Resume must-do changes before applying
- Add team scope explicitly: number of direct reports, total org size, and manager-of-managers responsibility.
- Add platform scale metrics: data volume, number of pipelines, number of source systems, business users supported.
- Add reliability metrics: uptime %, incident reduction, MTTR improvement, on-time delivery %.
- Add governance/compliance language: data quality controls, access governance, audit/compliance frameworks used.
- Add pharma domain proof: explicitly expand prior McKesson work with one or two outcome bullets.
- Keep AI/agentic section with measurable outcomes (hours saved, adoption, cycle-time reduction).

## 2) Biggest gap for this specific JD
- The role strongly prioritizes Azure stack depth (ADF, Synapse, ADLS, Event Hubs, Cosmos DB).
- Your resume is currently stronger in AWS + supply chain platforms.
- Mitigation strategy:
  - Add any real Azure work already done (even pilot/POC level) in bullets.
  - Start Azure Data Engineer Associate certification path immediately and mention "in progress".
  - Publish one architecture case study mapping your current platform design to Azure equivalents.

## 3) ATS optimization checklist (non-negotiable)
- Mirror job title in headline: "Senior Director, Data Engineering".
- Reuse exact keywords naturally: data governance, data security, enterprise data platform, real-time analytics, Databricks.
- Keep each role bullet in "action + scope + quantified impact" format.
- Keep date formats consistent and avoid dense paragraphs.
- Ensure LinkedIn and resume use matching title language and skill keywords.

## 4) Application strategy to improve callback rate
- Apply with referral first; direct applications alone have lower executive callback rates.
- Target 20-30 highly matched roles instead of broad volume.
- For each application, tailor top 8-10 bullets to that posting's language.
- Send a short recruiter note: 5 lines with role fit, scope led, and one signature metric.

## 5) Interview readiness assets
- Prepare a 1-page "Data Engineering Transformation Portfolio" with 3 case studies:
  - Platform modernization
  - Reliability/operations excellence
  - AI/agentic workflow adoption
- Prepare a 30-60-90 day plan focused on data platform stabilization, governance, and value delivery.
- Prepare leadership stories for: org scaling, executive conflict resolution, vendor decisions, budget trade-offs.

## 6) 14-day execution sprint
- Day 1-2: Finalize tailored resume and LinkedIn headline/about.
- Day 3-5: Add quantified AI/agentic outcomes and McKesson-domain bullets.
- Day 6-8: Build one Azure-aligned architecture case-study deck.
- Day 9-10: Start Azure certification and post "in progress".
- Day 11-14: Submit referral-led applications and recruiter outreach.
